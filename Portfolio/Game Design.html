Calling of the Werewolf

Description

- The this game is a small 2d platformer level made in unity. It was a project for my Game development class at ISU. The player must collect 10 coins and kill 5 enemies and then make it to the end goal in order to beat the game. Basic story of the game would be the player protecting the village from monsters.

Skills

- The game is programmed in C# and uses collection mechanics as it's main function. I have a understanding of the basics and fundamentals of C#. These skills were learned from my Game development class at ISU.

Lost space

Description

- Lost space is a 3d platformer prototype where the player shoots enemies,collect health orbs if needed, and platform across to the door that leads to level 2. Level 2 would consist of the player collecting parts and interacting with other game objects as a way to "repair" broken machines. They would also have to shoot enemies.

Skills

- Just like the first project the game uses C# as it's code. I was able to code a shoot mechanic using delta time, and was able to have a collection mechanic with the parts. I also was able to have a functioning health bar work based on touch from enemies or health orbs. I was also able to program basic enemy tracking to player.

CellBlocks Level Design

Description

- These are 2D levels from my capstone project at ISU. I was the level designer for this project and I designed the layout of the levels along with placements of enemies and puzzles.

-These levels serve as tutorial levels for the game and introduces several mechanics.

- In cellBlock1 the player first learns how to jump and swing over enemies which are the 2 lizards that are on the ground.

-Cellblock2 and CellBlock 3 introduces the player to buttons that they must make contact with. Once the number of buttons as been pressed the door opens for the player to leave the level. CellBlock 3 also requires the player to dodge enemies that fire projectiles at them.

CellBlock4 introduces the mechanic of putting boxes on buttons. The buttons now cannot be pressed by the player and instead the player left clicks on boxes to pick them up then thrwo them around onto buttons.

Skills

- These levels demonstrate my ability and understanding to design levels that help players be introduced to game mechanics.

- Creating puzzle mechanics that require the player to complete before they can move on to the next level.

- Knowing how to place enemies and obstacles strategically to where there is some challenge for the player but not impossible.

Hallways Level Design

Description

- These are more 2D levels from my capstone project at ISU. These levels are speedchecks where once the player presses a button they have a limited amount of time to get through the level while dodging enemies. If they don't make it in time the door closes and they must try again.

Skills

- These levels show my ability to create player urgency in a timed setting

- Shows my ability to create obstacles for urgent situations for the player to learn how to navigate through them.
